# Portafolio Management Module

## classes.py
File where all asked for classes and functionality was made

## main.py
LLM (chatgpt url: [conversation_link](https://chatgpt.com/share/69065828-bff4-800d-a344-9c6ca9ca11df)) created cli program to showcase the use case of the portfolio management

## How to run
- Run `python3 main.py` to start the cli program
- A list of options will appear with some already created Stocks
- To start, you could type the option `test` to create a Demo portfolio or you could type the option `1` to start creating your own portfolio
